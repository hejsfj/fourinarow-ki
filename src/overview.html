Überblick über die Projektstruktur

Paketüberblick

Das Java Projekt ist in sieben Pakete unterteilt. Alle graphischen Benutzeroberflächen und Treiberklasse sind im Package application gebündelt. Das Package AgentKI beinhaltet die Spiellogik für das Setzten der Spielsteine. In dem Package fileInterface ist die Kommunikation über die Dateischnittstelle realisiert. Parallel dazu wurde im pusherInterface die Kommunikation über den Pusher Server implementiert. Das Package gamefield umfasst die interne Repräsentation des Spielfeldes, in das die Spielsteine gesetzt werden. Im database Package befinden sich alle Datenbankfunktionalitäten. Zusätzlich befinden sich Hilfsklassen (z.B für die Pusherschnittstelle) im Package utils.

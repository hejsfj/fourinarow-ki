Überblick über die Projektstruktur

Paketüberblick

Das Java Projekt ist in sieben Pakete unterteilt. Alle graphischen Benutzeroberflächen und Treiberklasse sind im Paket application gebündelt. Das Paket AgentKI beinhaltet die Spiellogik für das Setzten der Spielsteine. In dem Paket fileInterface ist die Kommunikation über die Dateischnittstelle realisiert. Parallel dazu wurde im pusherInterface die Kommunikation über den Pusher Server implementiert. Das Paket gamefield umfasst die interne Repräsentation des Spielfeldes, in das die Spielsteine gesetzt werden. Im database Paket befinden sich alle Datenbankfunktionen. Zusätzlich befinden sich Hilfsklassen (z.B für die Pusherschnittstelle) im Paket utils.
